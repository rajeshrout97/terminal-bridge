Features

Why Terminal Bridge?

Built for developers who want their AI agent to control a remote Mac without SSH tunnels, port forwarding, or config files.

🤖

Model Agnostic

Works with Cursor, Claude CLI, Ollama, LM Studio, aider, or any tool that speaks HTTP, MCP, or stdio. No vendor lock-in.

🔒

Secure by Default

HMAC-SHA256 challenge-response auth, TLS encryption on all connections, and keys stored in macOS Keychain.

⚡

Zero Config for Cursor

The setup wizard auto-configures Cursor's MCP. Restart Cursor and your agent can immediately control the remote Mac.

🌐

LAN + Internet

Works over WiFi out of the box. Add Tailscale for free global access, or use the built-in relay server.

🔄

Background Service

The remote agent runs as a launchd service — auto-starts on login, survives reboots, zero daily maintenance.

📡

Six Interfaces

CLI, REST API, Python SDK, MCP server, stdio pipe, and interactive terminal. Use whatever fits your workflow.

Setup

Three commands. That's it.

Install on both Macs, run the setup wizards, done.

1

Install

Install Terminal Bridge on both Macs using pip or pipx.

pip install terminal-bridge
2

Setup remote

On the Mac you want to control. It generates a pairing code.

tbridge setup remote
3

Pair locally

On the Mac where your AI runs. Cursor MCP auto-configured.

tbridge setup local <CODE>
